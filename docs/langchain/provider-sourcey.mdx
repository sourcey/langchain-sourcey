> [Sourcey](https://sourcey.com) turns documentation, API references, and MCP
> servers into static sites.
> `langchain-sourcey` reads those published artefacts directly.

## Installation and setup

Install `langchain-sourcey`:

```bash
pip install -qU langchain-sourcey
```

No API key is required. Point the retriever at a public Sourcey docs root that
publishes `search-index.json`. If `llms-full.txt` is there, the retriever uses
it for full page content.

## Retriever

You can use [`SourceyRetriever`](/oss/integrations/retrievers/sourcey)
in a standard retrieval pipeline.

```python
from langchain_sourcey import SourceyRetriever
```

See the [usage example](/oss/integrations/retrievers/sourcey).

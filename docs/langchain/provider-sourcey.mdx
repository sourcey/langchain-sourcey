> [Sourcey](https://sourcey.com) turns documentation, API references, and MCP
> servers into static sites built for both humans and LLMs.
> `langchain-sourcey` lets LangChain retrieve directly from those published
> Sourcey artefacts.

## Installation and setup

`Sourcey` integration exists in its own partner package. Install it with:

```python
pip install -qU langchain-sourcey
```

No API key is required. Point the retriever at a public Sourcey docs root that
publishes `search-index.json`. `llms-full.txt` is recommended for full-page
hydration and stable citations.

## Retriever

You can use [`SourceyRetriever`](/oss/python/integrations/retrievers/sourcey)
in a standard retrieval pipeline.

```python
from langchain_sourcey import SourceyRetriever
```

See the [usage example](/oss/python/integrations/retrievers/sourcey).
